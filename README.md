# EN

# Netsoccer2

Netsoccer2 is a 2D multiplayer soccer game for Windows, focused on online matches, a social lobby, and competitive progression. This repository is used to publish client builds and update packages.

## What you will find in the game

- Public rooms, private challenges, and automatic matchmaking.
- Live match spectating, livescores, and rejoin support for active matches.
- Team system, invites, public/private/team chat, rankings, and Hall of Fame.
- Player profile with avatar, logo, history, XP, VIP rewards, and career statistics.
- Customization with fields, balls, cursors, kits, hairstyles, boots, and national teams.
- Client with fullscreen, windowed, and widescreen video modes, DirectX/OpenGL support, an audio panel, and FPS/ping display.

## Download and installation

1. Download the latest version from the **Releases** tab.
2. Extract the full package into its own folder.
3. Keep `Netsoccer2.exe`, `data/`, `locales/`, and the DLL files in the same folder.
4. Run `Netsoccer2.exe`.
5. If you already have an older installation, you can also use `patcher.exe`, which downloads the latest package and creates an automatic backup in `NS_Backups`.

## Default controls

- Movement: Mouse (cursor)
- Slide tackle: `W`
- In-match stats panel: `TAB`
- Clear chat: `F10`
- Anti-aliasing: `F11`
- Show FPS/Ping: `F12`
- FPS limiter adjustment: `PageUp` / `PageDown`

## Package contents

- `Netsoccer2.exe`: main client
- `patcher.exe`: updater with automatic backup
- `data/`: game assets
- `locales/`: language files
- DLLs and embedded components for audio, rendering, CEF, and Steam integration

## Important notes

- Windows client (Win32).
- Extract the full package; do not copy only the `.exe`.
- The first launch may ask for firewall permission for online connections.
- The official patcher downloads `Netsoccer2.zip` and replaces the files while preserving backups.
